# SPCXDOGE Meme Website V2

English meme-style official website inspired by the final SPCXDOGE logo/header visual direction.

## Sections

- Hero
- Story
- Token Details
- Tax & Rewards
- Live Data
- Why Hold LP
- Genesis Shareholder
- Roadmap
- Fan Art
- Links
- Enter DApp

## Animations

- Twinkling star background
- Breathing mascot
- Rotating moon
- Floating WOW/LFG/GM bubbles
- Rocket motion
- Infinite fan-art carousel
- Button hover effects

## Local Run

```bash
npm install
npm run dev
```

Open:

```bash
http://localhost:3000
```

## Build

```bash
npm run build
npm run start
```

## Deploy

Recommended: Vercel.

1. Upload this folder to GitHub.
2. Import the repo in Vercel.
3. Framework: Next.js.
4. Deploy.

## Replace Images

Main logo:

```text
public/logo.png
```

Fan art images:

```text
public/memes/
```
