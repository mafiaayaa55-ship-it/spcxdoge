# SPCXDOGE Meme Website V3.2

A clearer, MAME-style black/gold meme landing page for SPCXDOGE.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

Upload this folder to GitHub, import the repo in Vercel, and deploy with the default Next.js settings.

- Story

- Tax & Rewards
